# 🤝 Contribution Guidelines

Welcome to BiasLens! We are excited to have you build with us during GSSoC '26. To keep the project organized and fair for everyone, please follow these ground rules:

## 📝 Rules for Contribution
* **Issue Assignment:** Do not start working on an issue unless a Project Admin has explicitly assigned it to you. Pull Requests (PRs) from unassigned contributors will be closed.
* **How to Claim an Issue:** Comment on the issue you want to work on and briefly explain how you plan to implement the fix or feature. 
* **Pull Requests:** When submitting a PR, provide a clear description of your changes and link it to the issue by writing `Closes #IssueNumber` in the description.
* **Review SLA:** We review all issues and PRs within 24 hours. Please avoid tagging admins on personal channels.
